# 📊 Retail Sales Analysis Dashboard using Tableau

## 📌 Project Overview

This project presents an end-to-end Retail Sales Analysis Dashboard developed using **Tableau Desktop** and **Tableau Prep**. The objective is to transform raw retail sales data into meaningful business insights through data preparation, visualization, and interactive dashboards.

The project combines multiple regional datasets, cleans and prepares the data using Tableau Prep, and builds executive dashboards in Tableau Desktop for business decision-making.

---

# 🎯 Business Objective

The dashboard helps business stakeholders answer key questions such as:

- Which regions generate the highest sales?
- Which products contribute the most revenue?
- How do sales and profits vary over time?
- Which customer segments perform best?
- Which states require business attention?
- What are the major reasons for product returns?

---

# 📂 Dataset

The project uses retail sales data from multiple regions:

- Orders - Central
- Orders - East
- Orders - West
- Orders - South (2015)
- Orders - South (2016)
- Orders - South (2017)
- Orders - South (2018)
- Return Reasons

These datasets were merged and cleaned using Tableau Prep before visualization.

# 🚀 How to Use

1. Clone this repository.

```bash
git clone https://github.com/yourusername/Retail_Sales_Tableau_Dashboard.git
```

2. Open the Tableau workbook (`.twb`) using Tableau Desktop.

3. If prompted, reconnect the workbook to the datasets in the **Data** folder.

4. Explore the dashboards and interact with the visualizations.
